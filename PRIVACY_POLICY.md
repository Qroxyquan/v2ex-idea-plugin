# Privacy Policy

Last updated: March 6, 2026

This Privacy Policy applies to the **V2EX Reader for IntelliJ IDEA** plugin.

## What data this plugin processes

- Public V2EX content (topics, replies, and search pages).
- User-provided token values configured locally in IDE settings:
  - API Token (optional)
  - A2 Token (optional, for logged-in capabilities such as posting replies)

## What this plugin does NOT collect

- No source code from your local projects is uploaded by this plugin.
- No analytics, telemetry, or user behavior tracking is sent by default.
- No credentials are sent to third-party services except direct requests to:
  - `https://www.v2ex.com`
  - `https://www.google.com` (for in-IDE search pages)

## Data storage

- Configuration values are stored locally in IntelliJ settings on your machine.
- This plugin does not run a remote backend and does not store user data on external servers controlled by the plugin author.

## Data sharing

- Requests are made directly from your IDE to V2EX/Google endpoints as required by the feature.
- The plugin author does not receive or process these request payloads.

## Security notes

- Do not share your token values publicly.
- Revoke tokens from V2EX if you suspect compromise.
